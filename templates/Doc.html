Shotglass Documentation

Introduction

Shotglass is a simple web server framework that allows you to create web applications using Python.

Classes

Shotglass: The main class that represents the web server.

Methods

__init__(port=80, host="0.0.0.0"): Initializes the web server with a port and host.

route(path, request_type='GET'): Decorator to define a route for a specific path and request type.

errorhandler(error_code): Decorator to define an error handler for a specific error code.

start(): Starts the web server.

handle_request(): Handles incoming requests.

handle_get_request(client_socket, request): Handles GET requests.

handle_post_request(client_socket, request): Handles POST requests.

create_response(status_code, content_type, content, headers=None): Creates an HTTP response.

send_response(client_socket, response): Sends an HTTP response to the client.

run(): Starts the web server and begins handling requests.

Usage

  1. Create an instance of the Shotglass class.
  2. Define routes using the @route decorator.
  3. Define error handlers using the @errorhandler decorator.
  4. Start the web server using the start method.
  5. The web server will begin handling requests.
